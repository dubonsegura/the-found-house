How to publish on Hostinger:
1. Open Hostinger File Manager.
2. Go to public_html.
3. Back up the current files if desired.
4. Upload the ZIP for the current version.
5. Extract it.
6. Make sure index.html, shop.html, about.html, journal.html, contact.html, css/, js/, and images/ are directly inside public_html.

Do not upload the parent project folder itself unless you move the contents into public_html.
