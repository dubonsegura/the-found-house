The Found House Website Version 11 - Upload Instructions

1. Download and unzip this package.
2. In Hostinger File Manager, open public_html.
3. Back up your current files first.
4. Delete old website files from public_html.
5. Upload all files and folders from this package into public_html.
6. Make sure index.html is directly inside public_html.
7. Test every page on phone and desktop before turning the site live.

Recommended: keep this zip as Website_V11_Archive.
