Returns

A fair return policy for one-of-a-kind pieces.

Our goal is to be fair, clear, and practical.

Approved discretionary returns

If a return is approved for reasons other than damage, shipping error, or an inaccurate description, the customer is responsible for return shipping. Original shipping charges are generally non-refundable.

If we made a mistake

If an item was materially different from its description or we shipped the wrong item, The Found House will work with the customer to correct the issue, including return shipping when appropriate.

Shipping damage

If an order arrives damaged, please contact us promptly with photographs of the item, packaging, and shipping label. Timely photographs help support shipping insurance claims.

Condition requirement

Approved returns must be sent back in the same condition in which they were received, using appropriate protective packaging.
